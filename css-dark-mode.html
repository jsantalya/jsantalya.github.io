CSS ile Karanlık Mod

Neden?

Kullanıcı işletim sisteminde güzel güzel karanlık mod deneyimi yaşarken bembeyaz bir sayfaya girip göz spazmı yaşamak istemeyebilir. Tabi ki tasarım böyleyse yapacak fazla bir şey yok fakat, sayfanın tasarımcısı eğer kullanıcının dark modda olup olmadığını öğrenebilseydi bunun için bir şeyler yapabilirdi belki. 


            @media (prefers-color-scheme: dark) {
            // dark mod kodları buraya
          }

Kaynaklar:

  • Apple Human Interface Guidelines / Dark Mode
